This program has a single c++ file that will need to be compiled and executed. 
When the program is executed, it will create 10 random sets of data that will be
run through the subset sum, as well as the greedy subset algorithms and print out
the resulting weight value of each to the console as well as the time taken to execute
each algorithm. 

To run this program on siue servers:
g++ 'CS 456 P2'
./a.out
